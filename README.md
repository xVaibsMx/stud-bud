# 📚 Stud-Bud: Your Smart Learning Buddy 🤖

**Stud-Bud** is your personalized AI-powered study companion built to make learning easier, faster, and way more fun. Whether you’re preparing for exams or just curious about a topic, Stud-Bud helps you:

- 🍼 Understand complex concepts in simple terms
- ⏱ Revise anything in just one minute
- 🧠 Quiz yourself instantly to test retention
- 🤯 (Optional) Learn surprising fun facts
- 📜 (Optional) Track your study history

## 🔧 Tech Stack

- **Frontend:** React.js + Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI Power:** Gemini API (Google)

## 🚀 Why Stud-Bud?

Studying shouldn't be boring or confusing. Stud-Bud brings a smooth, chat-like experience that helps you stay focused, revise smartly, and actually _enjoy_ learning.
